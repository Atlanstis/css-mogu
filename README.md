# css 方案设计

## Setting 层

- 定义公共变量

- 公共变量：颜色、边框、字体大小、阴影、层级...

## Tools 层

- 定义常见的 css 片段，函数等

- [SassMagic 工具库](https://github.com/W3cplus/SassMagic)

## Base 层

- Generic 层：引入 [normalize.css](https://github.com/necolas/normalize.css)，重置浏览器默认样式

- Base 层：对各类元素基础样式进行补充

## Commonents 层

- 经典的组件：栅格、布局组件
- 自定义组件
